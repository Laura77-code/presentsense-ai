# Experiments

Experiments begin in Phase 2.

Planned emotion recognition experiments:

1. Custom CNN from scratch.
2. ResNet18 pretrained with frozen backbone.
3. ResNet18 pretrained with fine-tuning.
4. MobileNetV3 pretrained with fine-tuned classifier/head.

Metrics will include accuracy, macro precision, macro recall, macro F1-score, confusion matrix, and learning curves.
