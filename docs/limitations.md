# Limitations and Ethics

PresentSense is a visual communication feedback tool for presentation practice. It is not a medical, psychological, or clinical diagnosis system.

Current Phase 1 limitations:

- It only detects faces and draws overlays.
- It does not classify emotions yet.
- It does not estimate gaze or true eye contact.
- Detection quality depends on lighting, camera position, occlusion, and face visibility.
- Videos should be processed locally when possible to protect privacy.

Future phases will keep recommendations heuristic and supportive, not absolute judgments.
